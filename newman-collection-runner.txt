Newman - это command-line Collection Runner for Postman. 
Она позволяет запускать и тестировать Postman Collection прямо из командной строки.
Он построен с возможностью расширения, чтобы вы могли интегрировать его с серверами непрерывной интеграции (CI) и системами сборки.

0. Для того, чтобы запустить коллекции с помощью Newman, нужно сначала установить node.js.
Проверить, установлен ли у вас можно с помощью команды node --version

1. Устанавливаем newman
--> npm install -g newman (в командной строке)

Дальше нам нужно сохранить нашу коллекцию тестов, и тут есть два варианта - мы можем сгенерировать url для тестов или сохранить файлом.

Рассмотрим вариант с export, тогда мы будем работать с файлом - указывая путь к нему.
Скачивание .JSON файла коллекции осуществляется с помощью пункта “Export” после нажатия той же кнопки опций (…),
далее сохраняем файл на диск.

2.  Экспортируем коллекцию из Postman в какую-либо папку.
2.1 Экспортируем окружение для работы этой коллекции в ту же папку.

3. Для простоты (чтобы не прописывать длинные пути) переходим в ту директорию, где находятся коллекция и окружение.

4. Запускаем коллекцию с помощью команды newman run. 
Синтаксис команды следующий: 
--> newman run <collection-file-source> [options]

Например:
--> newman run HW-2-group-36.postman_collection.json -e HW-group-36.postman_environment.json
